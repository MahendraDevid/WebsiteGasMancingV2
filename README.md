# Website GASMANCING
### Deksripsi
Website GASMANCING adalah platform berbasis web yang memudahkan pemancing untuk menemukan lokasi pemancingan, menyewa alat-alat, membaca ulasan tempat pemancingan, serta menemukan informasi ikan dan tips memancing.

# Teknologi yang digunakan
| Kategori | Teknologi |
|-----------|------------|
| **Frontend** | HTML5, CSS3, JavaScript |
| **Library / Framework** | Vue |
| **Backend** | Nodejs|
| **Database** | MySQL |
| **Version Control** | Git + GitHub |
# Fitur Utama 
- Informasi Tempat Pemancingan: Detail Tempat, Lokasi, Fasilitas Pemancingan, Tarif, Ulasan Tempat, Foto, Ensiklopedia & Tips-tips memancing
- Pemesanan online tempat pemancingan & peralatan
- Ensiklopedia Ikan
- Tips Memancing
- Desain Responsif: Desktop & Mobile

# Installation
**1. Clone repository ini**
```bash
git clone https://github.com/MahendraDevid/WebsiteGASMANCING
```
**2. Install node modules yang diperlukan**
```bash
npm install
```
**3. Jalankan untuk melakukan pengembangan (Development Phase)**
```bash
npm run dev
```
**Atau jalankan ini untuk persiapan deployment (Production Phase)**
```bash
npm run build
```
